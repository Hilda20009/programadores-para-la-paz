Al enviar una petición POST a http://localhost:3000/registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con este JSON:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que el servidor recibe correctamente los datos enviados por el cliente, los procesa y devuelve una confirmación estructurada. Es útil para saber que la información llegó bien y fue registrada.
